eDiscovery

This was a large web application I worked on in a corporate setting. I fixed existing CSS bugs and I was also a front-end liason between design and development. Someone from design would create a mock-up for a module, I would create the HTML and CSS, and someone from tech would plug it into the existing application. It was enjoyable and intricate work.

I mainly used CSS/Sass and HTML for this project. I used Flexbox to fix a lot of the existing layout issues. I also worked a bit with JavaScript when applicable.
